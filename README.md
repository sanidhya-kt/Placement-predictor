# Placement Predictor 🎓

A simple Machine Learning web application that predicts whether a student will get placed based on their **CGPA** and **IQ score**.

## 🚀 Live Demo

Streamlit Web App: https://placement-predictor.streamlit.app

## 📊 Features

* Predicts student placement using a trained ML model
* Simple web interface built with **Streamlit**
* Takes **CGPA** and **IQ** as input
* Instant prediction result

## 🛠️ Tech Stack

* Python
* Scikit-learn
* Streamlit
* NumPy
* Pandas

## 📁 Project Structure

```
placement-predictor
│
├── app.py            # Streamlit web app
├── model.pkl         # Trained ML model
├── requirements.txt  # Dependencies
```

## ▶️ Run Locally

Install dependencies:

```
pip install -r requirements.txt
```

Run the app:

```
streamlit run app.py
```

Then open:

```
http://localhost:8501
```
